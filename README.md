# FUTURE-UIUX-02
Internship Task 02
Aura FITNESS – Mobile App UI/UX Design
Project Summary

Aura FITNESS is a conceptual mobile fitness application designed to provide users with an intuitive and engaging experience for workout planning, activity tracking, and exercise discovery. This project demonstrates core UI/UX design skills, including mobile-first design, user-centered layouts, visual hierarchy, and interaction consistency.

Objective

Design a complete mobile fitness application UI
Apply user-centered design principles
Ensure visual consistency and usability

Screens Designed

Login / Authentication Screen
Clean and minimal login interface
Mobile number and password input fields
Primary call-to-action: Get Started
Visually engaging fitness imagery

Home Screen

Personalized greeting with current date
Today’s workout plan highlight
Workout categories:
Beginner, Intermediate, Advanced, Expert / Pro
Training promotion card with ratings and discounted pricing
Bottom navigation for primary app sections

Activity & Progress Dashboard

Daily fitness summary including:
Steps, Calories burned, Workout count, Active hours
Circular activity progress indicator
Workout plan card with Start Workout action
Heart rate and hydration tracking
Weekly progress visualization
Challenge streak tracking

Exercises Listing Screen

Search functionality for quick exercise discovery
Category-based workout filters (Chest, Back, Legs, etc.)
Exercise cards containing:
Image preview
Short description
View details button
Favorite option
Designed for scalability and easy navigation

UX & Design Approach

Mobile-first UI design approach
Clear information hierarchy and layout structure
Consistent typography, spacing, and color system
Familiar fitness app interaction patterns
Optimized call-to-action placement for accessibility

Tools Used

Figma (UI Design & Prototyping)
Auto Layout
Grid-based alignment system
Freepik - for images
